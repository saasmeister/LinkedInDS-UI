# Doodle — a Style Pack for LinkedIn visuals

> Playful, hand-drawn, human. Rough sketch borders, solid offset shadows, a hand-lettered headline font, a warm-cream canvas, and doodle marks that feel sketched by a person — not generated by a machine.

This is a **style pack**: a look you switch on for a visual. It does **not** change the system's rules — see "What Doodle does NOT change" at the bottom. It restyles the archetypes you already use on the Visual Board.

The source of this pack was a UI/web design language (buttons, cards, modals, tables…). That scaffolding is irrelevant to a 1080×1350 post — so what follows is the doodle *aesthetic* translated to the canvas and the archetypes, nothing else.

## When to use it

- The client's voice is warm, human, founder-led, scrappy, anti-corporate.
- A post that wants to feel hand-made and approachable rather than slick and flat.
- "Make this fun / sketchy / hand-drawn / less corporate."

Not for: buttoned-up enterprise/finance brands, or when the client's brand layer is strongly defined and formal (ask first).

## How it's used

**You don't need to be told to use this.** A style pack is a capability the assistant reaches for whenever it sharpens a visual for the brand — or the user names it (*"do it in Doodle"*) or sets it as their **base style** (note it in `overrides/brand.css` / the brief so every visual starts here). Build the normal way (board, 3 variants, analogy-first); fold the doodle treatment into the chosen archetype.

**Doodle is a *skin* pack** — a surface treatment. It **composes** with a *composition* pack like **Bento**: e.g. Bento's tile mosaic rendered with Doodle's sketch borders + marks. Layer freely.

### Colour & font — agnostic
Its identity is the **technique** — the hand-drawn signature (sketch borders, solid offset shadows, doodle marks, slight tilt) — **not** a fixed palette or typeface. It **wears the brand layer**: body/labels in `--brand-font`, `--brand-primary` as the accent (the loop-underline, a key stat). The cream canvas named below is a **fallback** for when no brand is set (a brand-tinted warm neutral is just as valid), and the hand-lettered headline (Delius Swash Caps) is **optional flavour** — keep the brand font if that's the brand. What makes it Doodle is the treatment, which works in any colour or font.

## Visual foundations (the 1080×1350 canvas)

### Canvas
- **Background:** warm cream `#F9F6F0` (the doodle "paper"). The deep/section canvas is a darker cream `#EFEBE1` or the ink `#1F2937` for a chapter slide.
- **Texture:** a faint **dot-grid** behind content — `radial-gradient(#6B7280 1px, transparent 1px)` at ~22px spacing, ~8–12% opacity. It reads as graph/sketch paper. Keep it well under the safe margin so it never crowds text.
- Optional: a hand-drawn **crosshatch** or **wavy rule** as a divider between a header and the body — sketched, not a straight 1px line.

### Type
- **Headlines / focal numbers:** a hand-lettered display face — **Delius Swash Caps** (Google Fonts) is the reference. Load it and set the headline + big-stat in it. It is the single biggest signal of the doodle look.
- **Body / labels / attribution:** keep **legible** — the client's brand font, or a clean humanist sans. Do **not** set long body copy in the hand-lettered face; it hurts readability at feed size. Hand-lettered = headline + focal only.
- Weight 600+ for headlines; let the letterforms do the personality, don't over-tighten tracking.

### Ink, borders & the sketch radius
- **Ink colour:** near-black `#111827` (everything "drawn" — borders, marks, shadows derive from this).
- **Borders:** **3px solid ink** on every framed element (stat block, quote card, callout, image frame). Never hairline.
- **Hand-drawn radius** — never a uniform `border-radius`. Use one of three asymmetric values and rotate between them across sibling cards so no two adjacent frames match:
  - `255px 15px 225px / 15px 225px 15px 255px`
  - `15px 225px 15px 255px / 255px 15px 225px`
  - `225px 15px 255px / 15px 255px 15px 225px`

### Shadows — solid offset, zero blur
The depth signature. **No blurred shadows ever.** A solid ink offset:
- light frames: `box-shadow: 4px 5px 0 0 #111827;`
- bigger/feature frames: `6px 7px 0 0 #111827;`
- subtle chips/labels: `2px 3px 0 0 #111827;`

### Decoration — doodle marks (use the Icon Library!)
- Pull **doodle marks from the Icon Library** (`<icon-mark>` / `IconKit`) — star, sparkle, arrow, squiggle, spiral, loop-underline — and place them as **absolute accents** near the headline, the focal number, or the CTA. This is exactly what the Icon Library is for; do not hand-roll new SVGs.
- A **squiggle/loop underline** under one highlighted headline word is the signature move (use the brand accent colour for it).
- 1–3 marks per visual, max. They punctuate; they don't fill space.

### Motion / imperfection (static, but felt)
- Give framed cards and marks a **slight rotation** — `rotate(-2deg)` to `rotate(2.5deg)` — so nothing sits perfectly square. Alternate direction between siblings.
- This is the "hand-made" tell on a still image; keep it subtle (never beyond ~3°).

## Mapping onto the archetypes

- **Cover / headline (single):** cream canvas + dot-grid; hand-lettered headline, one word loop-underlined in the brand accent; a single doodle mark (star/arrow) near it; signature/handle in the clean body font. The analogy still leads — doodle dresses it, it doesn't replace the metaphor.
- **Big-stat:** the number in the hand-lettered face, huge; sits in (or beside) a 3px-ink sketch-bordered block with a solid offset shadow, rotated ~-1.5°; label in clean sans beneath.
- **Cards / comparison / 2×2:** each cell is a sketch-bordered card; **rotate the three radius variants** across cells and alternate the tilt; solid offset shadows; a doodle check/cross mark per cell where it helps.
- **Quote:** big hand-lettered quote on cream, a sketched quotation-mark doodle, attribution in clean sans; optional sketch-bordered portrait frame (rounded with a hand-drawn radius, 3px ink border, offset shadow).
- **Carousel:** cover uses the full doodle treatment; interior slides stay calmer (one sketch frame + one mark each) so the rail doesn't get noisy; keep the swipe-arrow footer. Per-slide content still vertically centred (GATE 7).

## What Doodle does NOT change

Doodle is a skin. The root `SKILL.md` still governs everything:

- **One Visual Board** — a doodle visual is still a `<section class="visual">` block on the single board, never a separate file.
- **3 variants** per round; the user picks one and you iterate into 3 more.
- **Analogy-led** — a single/quote visual is still built on one visual metaphor, never a stack of text.
- **Safe margins** (the 24px band) and **GATE 7** vertical centring still hold.
- **1080×1350**, brand layer still respected (folded in as the accent).

If the doodle aesthetic ever conflicts with a hard rule, the hard rule wins.
